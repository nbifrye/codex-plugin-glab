# GitLab Issues

Use the glab MCP server tools for all issue operations.

## List and search

- List open issues for the current or specified project
- Filter by labels, milestone, assignee, author
- Search issues by keyword in title or description
- Sort by created date, updated date, or priority

## Create

- Create issues with title and description
- Set labels, milestone, assignee, due date, weight
- Use issue templates when available in the project
- Link related issues

## Update and manage

- Edit title, description, labels, assignee, milestone, due date
- Add comments to issue threads
- Close or reopen issues
- Mark as confidential when needed

## Organize

- Add and remove labels
- Assign and unassign users
- Set milestone and due date
- Link issues to merge requests

## Free-tier notes

- Issue weights are available on Free tier
- Epics are a Premium feature -- use labels or milestones for grouping on Free
- Scoped labels (e.g., `priority::high`) are a Premium feature
- Issue boards are available on Free (one board per project)
- All other issue operations work on Free tier without limitations
